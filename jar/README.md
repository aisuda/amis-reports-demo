## 本地开启JSONQL服务
1. 需要安装java环境
2. 通过 java 命令启动JSONQL服务，执行：`java -jar ./aisuda-jsonql-server-version.jar`
3. 本地通过 http://localhost:8989/h2-console/ 可以查看数据库， JDBC URL 填 jdbc:h2:~/aisuda-model，用户名 root，密码空。
